Step 1 : Download these files and put in AWN folder

and create folder AWN in phone storage
place files in AWN folder

||Phone_Storage >> AWN >> downloaded_files 


i_usr Download(5MB)
fatmx Download(16MB)
RNX_OS Download(215MB)


Step 2 : Download and install app

RNX IO Download(1.5MB)


Step 3 : Give storage permission to RNX IO App

then open app and wait for installation


BUG : if you see dummy output in audio then restart RNX IO App




Step 4 : Install Done | How to use |

root password is == 1234


toggle keyboard == press volume up key

scroll == swipe touch screen

left click = single touch
drag mouse left click == touch and hold then swipe touch screen

right clich == press volume down key then single touch
drag mouse right click == press volume down key then touch and hold then swipe touch screen


turn up android volume == single touch anywhere on screen then press volume up key
turn down android volume == single touch anywhere on screen then press volume down key

drag window == press volume down key then swipe touch screen
