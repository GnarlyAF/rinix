Step 3 : Installing Linux



	install Latest binaries on RNX Core via RNX Neo
	run below command


pingo get




	install linux
	run below command


rnx get-default-linux



	then set your linux in promt
	1) root password
	2) user setup


to Start basic services 
	run them as root user

run command 
"rnx_uwax enable xio"
on default display :1
xio is GUI Display for RNX
to enable RNX XIO app on startup

run command 
"rnx_uwax enable proftpd"
default port 7021
proftpd is ftp server use to transfer file from other devices
to enable proftp server on startup

run command 
"rnx_uwax enable sshd"
default port 7022
sshd is secure shell server use to open terminal other device
to enable sshd server on startup

run command 
"rnx_uwax enable vncd"
default port 5972 on default display :72
vncd is remote desktop use to connect display from other device
to start vnc so you can use RNX display on other devices
 on startup

Guide for How to use RNX XIO
---------

	Index - Table
	
toggle keyboard == press volume up key

scroll == swipe touch screen

left click = single touch
drag mouse left click == touch and hold then swipe touch screen

right click == press volume down key then single touch



turn up android volume == single touch anywhere on screen then press volume up key
turn down android volume == single touch anywhere on screen then press volume down key

drag window == press volume down key then touch and hold then swipe touch screen
