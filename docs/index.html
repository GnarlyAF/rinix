----------- Rinix OS - Installation Guide---------

	Rinix OS is a Desktop GNU/linux on android kernel(custom linux kernel)
	based on debian using proot 
	supports grapical , command line , audio , microphone

	using termux , termux-api apps 

	for audio,mic we use pulseaudio modules

	for video output you can use vnc client 
		we are currenty working on native video output without using tcp,sockets
		and android cam as webcam
		




android apps in photo for vnc is
	bvnc

linux x11 themes in photo is
	papirus-icon-theme
	breeze-cursor-theme
	adwaita dark xfce4 style
	xfce4
	xfce-terminal with custom settings


Features :-
	audio,microphone support on proot linux distro
	auto boot with termux app(optional)
	full x11 support with openGL
	
Issue :-
	for any issue visit Here
	
--------- Steps for installation -------------

Step 1 :

	download "termux" and "termux api"
	from play store
	open termux and type


apt update
apt install pulseaudio sox nano tar termux-api wget


	configure pulse audio in termux


cd
nano ../usr/etc/pulse/default.pa


		add these lines in nano at last in file ../usr/etc/pulse/default.pa
		"Ctrl + x" to exit nano test editor 
		192.168.0.0/16 is for LAN ip addresses


#### awnto edits
load-module module-native-protocol-tcp auth-ip-acl=127.0.0.1;192.168.0.0/16 auth-anonymous=1
load-module module-cli-protocol-unix
load-module module-sles-source
load-module module-sles-sink
#### end awnto edits




	configure pulse audio in termux


cd
nano ../usr/etc/pulse/daemon.conf


		change "; exit-idle-time = 20" ---> " exit-idle-time = -1" don't remove first space


Step 2 :

	to install debian root filesystem


cd
mkdir rinix-fs && cd rinix-fs
mkdir debian && cd debian

pkg install wget openssl-tool proot -y && hash -r
wget https://raw.githubusercontent.com/EXALAB/AnLinux-Resources/master/Scripts/Installer/Debian/debian.sh
bash debian.sh



Step 3 :

	login to debian proot


./start-debian.sh


	then install needed softwares for Rinix OS


apt install nano wget xfce4 pulseaudio openssh-server proftpd tigervnc-standalone-server tigervnc-xorg-extension tigervnc-viewer tigervncserver-common dbus-x11 pulseaudio


Step 4 :
	configure softwares for Rinix OS
	
	setting correct permission


chmod 4755 /bin/su
	

	ports 0 - 999 is reserved for root
	


nano /etc/ssh/sshd_config

	change "#Port 22" ---> "Port 7022" remove '#' also 




nano /etc/proftpd/proftpd.conf

	change "Port 21" ---> "Port 7021" 




nano /etc/pulse/default.pa

		add these lines in nano at last in file /etc/pulse/default.pa


#### awnto edits
load-module module-tunnel-sink-new server=127.0.0.1 sink_name=Awnto_spk sink_properties=device.description=Awnto_Wireless_Speaker
load-module module-tunnel-source-new server=127.0.0.1 source_name=Awnto_mic source_properties=device.description=Awnto_Wireless_Microphone
#### end awnto edits


	set pulse config for Rinix OS


nano /etc/pulse/daemon.conf

		change "; exit-idle-time = 20" ---> " exit-idle-time = -1" don't remove first space
	

	create a user >> use your desired username in place of "rishu" 


adduser rishu


	set vnc password for user


su rishu
vncserver :78
vncserver --kill :78

	if ~/.vnc/xstartup not exit then l 
	"nano ~/.vnc/xstartup" 
	add "xfce4-session" as new line
	and chmod 755 to xstartup
	connect to vnc
	your local ip is 127.0.0.1
	192.168.x.x lan ip
	use "ip addr" to know your ip

	configuration of softwares for Rinix OS is done -- exit to debian proot 	


exit



Stpe 5 : Install Rinix OS

	give storage permission to termux app to share storage file


mkdir /sdcard/AWN


	give microphone permission to termux-api app if you want microphone support for Rinix OS


	install Rinix OS


cd
wget http://rinix.awnto.com/files/up-rinix
chmod 755 up-rinix
sh up-rinix


	config Rinix OS


nano rinix/start

	replace "RINIX_RDU=rishu" to your username "RINIX_RDU=username"
	replace "RINIX_RFS=xxxxxx" to "RINIX_RFS=yyyyy" to your debian proot path
	
	in RINIX OS
		rinix_rbin		/root/orc
		rinix_user		/home/rishu/orc
		shared_folder		/home/rishu/AWN
		android_storage		/home/rishu/sdcard
		other_shared		/home/rishu/shared/*
	here rishu stands for your username eg /home/user/AWN
	
	defaults
		DISPLAY=:78 over vnc port 5978
		
Step 6 :

	Start RinixOS


cd
sh rinix/start



Step 7 : to make Rinix OS auto start when termux app open
	
	this is a option step (recommanded to use Step 7)
	login to termux shell (not Rinix nor Debian shell)


cd 
nano ../usr/etc/bash.bashrc


	add this line at last
	"sh /data/data/com.termux/files/home/rinix/start"
	

Step 8 : enjoy

Step 9 :
	to stop audio welcome in Rinix OS


cd
nano .rinix/rinix/orc/lock

	add '#' at starting of line
	where termux-tts-speak comes
	in termux shell

Solution for libreoffice error in Rinix OS

	You can edit file /usr/lib/libreoffice/program/oosplash with any 
	HEX editor (bless for example). Just find text "/proc/version" 
	(LibreOffice checks this file) and change to for example "/prod/version". 
	Then you can create fake empty "/prod/version" file in the filesystem, 
	then Office starts and works well 
	
	fake proc/version file Here
	
Solution for chromium

	use "chromium --no-sandbox" instead of "chromium"
	or you can use firefox
